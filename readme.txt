Bing Wallpaper Installer
=======================

Thank you for installing the Shawn's Bing Wallpaper Automation!

- Your wallpapers will be downloaded and set automatically each day.
- You can change the image quality and archive retention size by editing the config.ini file in the installation directory.
- To manually run the updater, use the shortcut in your Start Menu or run the PowerShell script directly.
- For support or updates, contact the author or visit the project page.

Enjoy your new wallpapers, with no more BING annoyances :)
